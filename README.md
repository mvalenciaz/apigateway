# apigateway
Este es un proyecto de ejemplo de api gateway con Node.js

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm run start
```

## Rutas

### Users
Ver todos los post del usuario
```bash
GET /users/:userId/posts
```

### Posts
Ver todos un post con su usuario
```bash
GET /posts/:postId/
```
